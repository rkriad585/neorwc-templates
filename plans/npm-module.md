# Plan: npm Module Docs
**Goal:** Create a documentation suite for an npm library.

**Required Files:**
1.  `README.md`:
    - Badges (npm downloads, version).
    - Installation (`npm i package`).
    - API Table (Function | Params | Returns).
2.  `docs/API.md`: Deep dive into exported functions.
3.  `docs/CHANGELOG.md`: Version history following SemVer.
