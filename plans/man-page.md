# Plan: Single Man Page
**Goal:** Create a single file that looks like a UNIX manual entry.

**Required Files:**
1.  `docs/man/neorwc.1.md` (or project name).

**Instructions:**
- Use strictly uppercase headers (SYNOPSIS, OPTIONS).
- Do not use "I" or "You". Use passive voice ("The file is written...").
- List every single command line argument found in the code.
- Provide 3 concrete examples in an EXAMPLES section.
