# Plan: Standard README Overhaul
**Goal:** Create a single, comprehensive `README.md` file in the root directory.

**Requirements:**
1.  **Title & Description:** Clear name and one-line pitch.
2.  **Features:** Bullet points of top 5 capabilities.
3.  **Installation:** `npm/pip/cargo` install commands.
4.  **Usage:** A "Hello World" code snippet showing how to run it.
5.  **Configuration:** A table of environment variables or CLI flags.
6.  **License:** State the license type.

**Output:** Only `README.md`.
