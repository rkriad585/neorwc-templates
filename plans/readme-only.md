# Plan: Single README Generation

**Goal:** Create **ONLY** a `README.md` file. Do not create any other files.

**Required Files:**
1.  **`README.md`** (in root).

**Content Structure:**
1.  **Header:** Project Name + Dynamic Badges (License, Language, Version).
2.  **Elevator Pitch:** One sentence describing what this code does.
3.  **Features:** Bullet points of technical capabilities found in the code.
4.  **Installation:** `npm install`, `pip install`, or `go get` command.
5.  **Usage Example:** A code block showing the most common use case.
6.  **License:** A footer mentioning the license type.

**Constraint:**
- If the file is longer than 500 words, summarize it. Keep it punchy.
