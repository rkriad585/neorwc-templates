# Plan: PyPI Package Docs
**Goal:** Create a standard Python package documentation suite.

**Required Files:**
1.  `README.md`:
    - Project Title & Badges.
    - `pip install <package_name>`
    - Quick Start (4-5 lines of code).
    - Configuration options.
2.  `docs/usage.md`: Detailed examples using `import module`.
3.  `docs/CONTRIBUTING.md`: Instructions for using `black`, `flake8`, and `pytest`.
